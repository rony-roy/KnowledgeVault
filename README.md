Navigation links in the middle
Action button on the right

Hero / Banner Section:
Background image
Main headline and description
Call-to-action buttons

Notes Structure Section:
Four informational cards
Icons/images with descriptions

Brain Structure Section:
Image on the left
Explanation content on the right

Early Access Section:
Call-to-action for early users
Center aligned message and button

Footer:
Company logo
Social media links
Copyright notice
